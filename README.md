# Incubator

一个技术孵化器，这里包含杂乱的，不成熟的，实验性质的简单实现。出于探索目的，记录在这边，如果其中的某个想法发展起来了，可能会移到一个单独的代码库。

## 目前的实现

### observable

一个简单的，不可 compose 的 observable

### simple platformer

一个简单的平台游戏实现，随机生成游戏场景，可以移动游戏角色。

### snappy box

类似 Flappy Bird, 只不过这边用一个矩形来代替游戏角色。
